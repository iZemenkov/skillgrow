# Система рекомендации недостающих навыков для карьерного роста в области DS

SkillGrow — это система анализа вакансий и рекомендаций недостающих навыков для карьерного роста в области Data Science.  
Проект собирает вакансии с [hh.ru](https://hh.ru/) через публичное API, анализирует требования, выделяет ключевые навыки и сопоставляет их с данными из резюме пользователя.  
На основе этого формируются рекомендации по развитию компетенций.

## Основные возможности

- Парсинг вакансий с hh.ru по ключевым запросам (Data Scientist, Analyst и др.)
- Очистка и агрегация данных в мастер-датасет
- Анализ текстов вакансий (EDA, извлечение навыков, кластеризация)
- Применение языковых моделей (ruBERT / DeepSeek) для выделения и группировки навыков
- Рекомендательная система по недостающим навыкам
- Streamlit-приложение для интерактивной работы и загрузки резюме
